# README

Simple script, which downloads the images you want, using the unsplash api

![](https://source.unsplash.com/random)

## Use It

1. Install the dependencies:

```bash
pnpm i
```

2. Run the script:

```bash
pnpm download
```

3. Open images directory

```bash
pnpm see:win

or

pnpm see:mac

or

pnpm see:lix
```
